λ-Cálculo
MT Não Determinística
No λ-cálculo só existe função. Números (numerais de Church), verdadeiro/falso e até a recursão (combinador Y) são funções. Calcular = repetir uma única regra, a β-redução (substituir o parâmetro pelo argumento), até não dar mais para simplificar. Cada linha abaixo é um passo dessa redução.
MULT 2 3 AND TRUE FALSE ISZERO 0 PRED 3 FACT 3 (\x. x x) (\x. x x)
A máquina aceita palavras que são uma metade repetida (ww): aa, abab, aabaab. Como ela é não determinística, ela adivinha onde termina a 1ª metade (cada palpite vira um ramo) e depois confere se as duas metades batem, posição a posição. Aceita se algum ramo chegar em qAccept; rejeita se todos morrerem.
aa abab aabaab abba aba abc
